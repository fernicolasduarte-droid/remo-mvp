REMO MVP - PAQUETE LISTO PARA RENDER

Este paquete ya tiene la estructura correcta para subir a GitHub y desplegar en Render.

Estructura:
- package.json
- src/server.js
- public/index.html
- README.txt
- README_RENDER.txt

Configuracion Render:
- New + > Web Service
- Repository: el repo donde subas estos archivos
- Root Directory: dejar vacio
- Runtime: Node
- Build Command: npm install
- Start Command: npm start
- Plan: Free para probar

Despues Render te va a dar una URL HTTPS tipo:
https://remo-mvp.onrender.com

Con esa URL probamos ubicacion actual desde el celular.
